# v0.1.0

Initial public prototype release for AI Dynamic Island on macOS.

## Highlights

- Unified notch-aware dynamic-island shell for 14-inch MacBook Pro layout constraints.
- Pixel mascots and motion states for Codex and Claude Code.
- Expand/collapse interaction model with hover and pinned card states.
- Live Codex + Claude monitor pipelines feeding task rows, models, and context usage into UI.
- Sunset strategy for stale threads:
  - `<= 2m` keep live state
  - `2m ~ 15m` degrade to idle
  - `> 15m` hide row
  - `> 30m` mark status unavailable
- Regression coverage for monitor parsers, monitor-to-UI presentation chain, shell geometry, and fallback rendering.
